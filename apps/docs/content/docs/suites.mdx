---
title: Suites
description: Run a group of tests together.
---

A **suite** is a saved group of tests you want to run together — say, every test for a release.
Instead of running them one by one, you run the suite once and get a single combined report.

You can run a suite against **one or more environments** at the same time, so you can check, for
example, staging and production in a single go.

<Callout>
  Only live tests can be in a suite — tests Claude created stay out until you've
  [promoted them](/docs/review-queue).
</Callout>
